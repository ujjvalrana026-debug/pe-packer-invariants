# malware_forensics_
Static forensic framework for attributing polymorphic malware without execution or unpacking. Identifies 13 PE structural invariants across 9 families (Sality, Emotet, Trickbot, Formbook, Lumma + more). 362-sample dataset, Wilson CIs, BH-corrected stats, XGBoost validation on 100k SOREL-20M samples (AUC 0.9961).
